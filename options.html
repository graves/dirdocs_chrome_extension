About

This extension swaps GitHub directory commit messages with descriptions from .dirdocs.nuon when available.

Use the popup on a repo page to enable/disable and load mappings per repo.

Tips

  • Upload a local .dirdocs.nuon to override.
  • “Fetch from repo” reads the default branch (HEAD).
  • Hover any replaced message to see full text in a custom tooltip.
  • Generate a .dirdocs.nuon file using dirdocs
